사용자에게 1~100 사이의 값을 입력을 받아
아래 성적을 체크한 뒤 성적 결과를 출력해주도록 작성
유효하지 않은 값 입력 시 값을 체크 후
"잘못된 값을 입력하셨습니다. 1~100 사이 숫자를 입력해주세요"
문장 출력 후 다시 값 입력 대기 상태가 될 것.

90 A+
80 A
70 B+
60 B
50 C+
40 C
40 미만 F

결과 당신의 학점은 0입니다. << 문장으로 출력할 것.

출력 후 다른 학점을 조회하시겠습니까?
(1을 입력하면 추가로 학점을 입력하는 상태로 대기)
(2를 입력하면 프로그램 종료)
상태로 대기 할 것/ 모든 상황에 사용자에게 안내 문구를 출력할 것.

#include <iostream>
using namespace std;

void grade();
void restart();

int main() {
	grade();
}

void grade() {
	cout << "성적을 입력해주세요";
	int score;
	cin >> score;

	if (score < 0 || score > 100) {
		cout << "잘못된 값을 입력하셨습니다. 1~100 사이의 숫자를 입력해주세요." << endl;
	}
	else if (score >= 90) { cout << "당신의 학점은 A+입니다." << endl; }
	else if (score >= 80) { cout << "당신의 학점은 A입니다." << endl; }
	else if (score >= 70) { cout << "당신의 학점은 B+입니다." << endl; }
	else if (score >= 60) { cout << "당신의 학점은 B입니다." << endl; }
	else if (score >= 50) { cout << "당신의 학점은 C+입니다." << endl; }
	else if (score >= 40) { cout << "당신의 학점은 C입니다." << endl; }
	else
		if (score < 40) { cout << "당신의 학점은 F입니다." << endl; }

	restart();
}

void restart() {
	// 출력 후 다른 학점을 조회하겠습니까?
	cout << "출력 후 다른 학점을 조회하겠습니까?";

	// 1을 입력하면 추가로 학점을 입력하는 상태 대기
	int select;
	cin >> select;
	if (select == 1) {
		// grade()함수를 호출하겠다.
		grade();
	}
	else if (select == 2) {
	}
}
